Q1. Difference between var, let and const keywords in JavaScript?

Ans: var is the traditional way of declaring variables in JavaScript and it is globally scoped, it is considered a bad practice in JavaScript to declare variables with var nowadays. let and const was introduced as a new way to declare variables with ES6. With let, you may reassign the value but the variable will only be used inside the block it was defined in and not outside, kind of like having its own scope. you cannot reassign the value if you declared the variable as const.

Q2. Difference between map, filter, forEach and find?

Ans: map returns an array with the same length,
filter as the name implies, it returns an array with less items than the original array
find returns the first items in an array that satisfies a condition
forEach Introduced in ES5. Given an array, you can iterate over its properties using list.

Q2. Why use template strings in Javascript?

Ans: Template literals provide an easy way to interpolate variables and expressions into strings. The method is called string interpolation.
